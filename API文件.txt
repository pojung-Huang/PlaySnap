API 文件說明
===========

【基本配置】
- Python 3.x
- Flask
- SQLite 資料庫

【API 端點】

1. 獲取排行榜
   GET /api/leaderboard
   
   響應格式：
   [
     {
       "name": "玩家名稱",
       "score": 分數,
       "date": "YYYY-MM-DD HH:MM"
     }
   ]

2. 提交分數
   POST /api/leaderboard
   
   請求格式：
   {
     "name": "玩家名稱",
     "score": 分數
   }

   響應格式：
   {
     "success": true/false
   }

【資料庫結構】
表名：Score
欄位：
- id: INTEGER PRIMARY KEY
- name: VARCHAR(50)
- score: INTEGER
- date: DATETIME

【安全性考慮】
1. 輸入驗證
   - 玩家名稱長度限制：50 字元
   - 分數必須為正整數
   - 防止 SQL 注入

2. XSS 防護
   - 過濾特殊字元
   - 轉義 HTML 標籤

3. CSRF 防護
   - 使用 Flask 內建的 CSRF 保護

【錯誤處理】
1. 400 Bad Request
   - 無效的請求格式
   - 缺少必要參數

2. 500 Internal Server Error
   - 資料庫操作錯誤
   - 伺服器異常

【部署注意事項】
1. 確保 Python 3.x 環境
2. 設置正確的檔案權限
3. 配置資料庫連接
4. 設置適當的快取策略
